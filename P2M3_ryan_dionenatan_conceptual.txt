2. Jelaskan kapan harus menggunakan NoSQL dan Relational Database Management System !

RDBMS dipakai bila datanya bersifat relasional, mempunyai struktur yang tetap dan bila datanya harus memiliki bentuk yang konsisten.
NoSQL dipakai bila data yang dimasukkan bersifat fleksibel, bila data butuh diakses dan dimasukkan secara cepat dan bila datanya tidak mempunyai struktur tertentu.

3. Sebutkan contoh 2 tools/platform NoSQL selain ElasticSearch beserta keunggulan tools/platform tersebut !

MongoDB dan Apache CouchDB.

Keunggulan MongoDB :
MongoDB menyimpan data dalam format BSON, sehingga MongoDB dapat menyimpan data dalam jenis struktur data apapun. Dan data bisa replikasi secara real-time secara cepat.

Keunggulan Apache CouchDB :
CouchDB memiliki toleransi kesalahan yang tinggi, sehingga data dapat diproses secara aman. Dan mendukung HTTP API sehingga interaksi data antar aplikasi jadi lebih gampang.
